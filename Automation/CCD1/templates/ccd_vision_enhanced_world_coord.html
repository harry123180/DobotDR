內外參檔案要求說明 (v4.1)

內參檔案（嚴格命名）：
  • camera_matrix_YYYYMMDD_HHMMSS.npy
  • dist_coeffs_YYYYMMDD_HHMMSS.npy
  • 檔案必須成對存在且時間戳一致
  • 內參矩陣格式: 3×3
  • 畸變係數: 至少4個參數
外參檔案（較寬鬆命名）：
  • *extrinsic*.npy (包含extrinsic字串)
  • 包含rvec和tvec的字典格式
  • rvec: 3×1旋轉向量
  • tvec: 3×1平移向量
操作步驟： 1. 將NPY檔案放入程式同層目錄 2. 點擊「掃描檔案」確認檔案 3. 點擊「確認導入」載入標定數據 4. 確認轉換器狀態為「已啟用」 5. 世界座標轉換為Z=0平面投影，精度保留2位小數

保護範圍說明

功能說明：
  • 啟用後，只有在指定範圍內的物件才會被識別
  • 超出範圍的物件將被過濾，不會出現在檢測結果中
  • 需要載入世界座標標定數據才能使用此功能
  • 關閉時，所有檢測到的物件都會被顯示
座標範圍設定：
  • X軸: -122 ≤ X ≤ -4 (mm)
  • Y軸: 243 ≤ Y ≤ 341 (mm)
  • 單位：毫米 (mm)
  • 精度：支援到小數點後2位

運動控制握手邏輯說明 (v4.1)

  1. 系統初始化完成 → Ready=1，準備接受控制指令
  2. PLC下達控制指令 → 系統檢查Ready=1才執行
  3. 開始執行操作 → Ready=0, Running=1
  4. 操作執行完成 → Running=0
  5. PLC清零控制指令 → Ready=1，準備下次操作
  6. 異常發生時 → Alarm=1, Initialized=0
  7. v4.0: 世界座標轉換 → 需要有效的內外參數據才能啟用
  8. v4.1: 保護範圍過濾 → 僅有效範圍內物件會被識別
